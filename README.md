## Overview
The **NASA data explorer & Space Image Analyzer** is a web application that allows users to explore NASA's image database  using   AI-powered features for  better user experience, including natural language search and image analysis.
This application is using NASA's Open APIs to fetch images and information, and uses OpenAI's GPT-4o mini model to provide detailed scientific explanations of the images. The frontend is built with React and Vite, while the backend is powered by Node.js and Express.
Using monorepo structure, the project is divided into `frontend/` and `backend/` directories for better organization and maintainability.For this coding challenge, the focus is on building a functional prototype with essential features, while future improvements will be made to enhance the application further.

## Features
- **Astronomy Picture of the Day (APOD):** View NASA's daily featured image with AI-generated analysis.
- **AI-Powered Image Analysis:** Analyze NASA's images with detailed scientific explanations generated by OpenAI.
- **Natural Language Search:** Search NASA's image database using natural language queries.
- **Responsive Design:** Optimized for both desktop and mobile devices.

## Tech Stack
- **Frontend:** React, Vite
- **Backend:** Node.js, Express
- **APIs:** NASA Open APIs, OpenAI API
- **Testing:** Jest, Supertest

## Installation

### Prerequisites
- Node.js (v16 or later)
- npm or yarn
- OpenAI API Key
- NASA API Key

### Backend Setup
1. Navigate to the `backend/` directory:
   ```bash
   cd backend
   ```
2. Install dependencies:
   ```bash
   npm install
   ```
3. Create a `.env` file in the `backend/` directory and add the following:
   ```env
   OPENAI_API_KEY=your_openai_api_key
   NASA_API_KEY=your_nasa_api_key
   FRONTEND_URL=http://localhost:5173
   ```
4. Start the backend server:
   ```bash
   npm start
   ```

### Frontend Setup
1. Navigate to the `frontend/` directory:
   ```bash
   cd frontend
   ```
2. Install dependencies:
   ```bash
   npm install
   ```
3. Start the development server:
   ```bash
   npm run dev
   ```

## Usage

### Home Page
The home page provides an intuitive interface to access all main features of the application:
- Quick access to APOD (Astronomy Picture of the Day)
- Direct link to Image Search functionality
- Dynamic background showcasing space imagery

### APOD (Astronomy Picture of the Day)
1. Navigate to the APOD page
2. View NASA's featured astronomical image of the day
3. Click "Get AI Analysis" to receive detailed scientific explanations
4. The analysis includes:
   - Technical details about the astronomical object
   - Scientific significance
   - Historical context when relevant

### Image Search
1. Enter natural language queries in the search bar
   - Example: "Show me pictures of Mars rovers from 2020"
   - Example: "Jupiter's moons images from 2024"
2. The search functionality supports:
   - Year-specific queries
   - Object type filtering
   - Natural language processing for better results
3. Results are displayed in a responsive grid layout

## Performance Optimization

### Backend Optimizations
1. **Caching System**
   - Implements in-memory caching for NASA API responses
   - Cache invalidation after 1 hour
   - Reduces API calls and improves response times



### Frontend Optimizations
1. **State Management**
   - Efficient React state management
   - Debounced search functionality
   - Optimized component rendering


### OpenAI Token Usage
- Tracks usage of OpenAI API tokens for cost management and optimization
- Stores timestamp and token count for each API call

### Cache
- Implements caching mechanism for NASA API responses
- Stores search results with timestamps for efficient retrieval
- Implements automatic cleanup for expired cache entries

### Search suggestions
- Provides search suggestions based on user input
- Uses hardcoded suggestions for common queries
